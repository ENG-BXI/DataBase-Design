بيانات أساسية للطالب:

ID → رقم فريد لكل طالب (UUID أو رقم مسلسل)

الاسم الكامل → الاسم الأول واسم العائلة

تاريخ الميلاد → علشان نعرف عمر الطالب

الجنس → ذكر أو أنثى

البريد الإلكتروني → للتواصل وإشعارات النظام

رقم الهاتف → للاتصال السريع

العنوان → عنوان الطالب الحالي أو دائم

بيانات السكن:

الغرفة/السكن الحالي → رقم الغرفة أو الوحدة السكنية

تاريخ الدخول للسكن → متى بدأ الطالب يسكن

تاريخ الخروج (لو موجود) → متى انتهت مدة السكن أو غادر

بيانات مالية:

الرسوم المدفوعة → كل دفعة دفعها الطالب

الرسوم المتبقية → أي مبلغ باقي عليه
